注意事項

第一重活動：

1. 參加抽獎活動之不同張發票恕不累積發票金額。

2. 現金抵用券、商品兌換券序號使用期限至2020/2/15，逾期恕無法兌換，每券限使用乙次。

3. 加價購商品序號使用期限至2020/2/15，逾期恕無法使用，每券限使用乙次。

4. 本券須於下一筆消費使用，恕不兌換現金或找零，也不列入會員消費累積。

5. 本券不限門市使用，僅限於棉花田實體門市使用。

6. 本券一經使用不得退換貨或要求補發，抵用券/兌換券/加價購商品序號，數量有限，送完為止。

7. 基於個人資料保護法保護個資原則，棉花田生機園地決不外洩參與活動者之個人資料。惟活動者於參加本活動前了解並同意主辦單位於本活動之範圍內，得蒐集、處理及利用本人之個人資料(包括但不限於姓名、e-mail、手機、會員卡號等)。

8. 活動辦法若有未盡事宜，棉花田保有隨時解釋、變更活動辦法與獎品內容，以及暫停、延長、提前終止本活動之權利。並公告於本公司活動網站，將不另行通知。


第二重活動：

1. 凡活動期間單筆消費折扣後每滿588元以上，並於活動網站登錄有效購買發票及個人資料(請保留發票實體正本及購買明細)，即可參加抽獎，有機會抽中飛利浦時尚家電、棉花田禮券。獎品不得更換，亦不可更換其他商品或兌換現金。

2. 將於2020/2/5抽獎，並公告於棉花田官方FB。

3. 機會中獎活動，依所得稅法之規定，須列入個人年度所得，得獎人須依中華民國稅法規定辦理扣繳事宜。未滿20歲之得獎者應由其法定代理人代為領獎。

4. 得獎人須依規定填寫並繳交相關發票正本、身分證影本；若中獎者不願意提供相關資料，視同放棄中獎資格。

5. 活動參加者需保證所有填寫或提出之資料均為真實且正確，並未冒用或盜用他人資料，如有不實或不正確資訊，本公司將取消其中獎資格，並就其損害本公司或任何第三人權益，保留一切民、刑事訴追之責。

6. 商品圖片僅供參考，請以實際商品為主。本公司保有活動修改之權利。
